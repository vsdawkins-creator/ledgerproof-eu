# LedgerProof Admin Console

Read-only Next.js dashboard for pilot customers. Lists receipts issued by your publisher_id; searchable; clickable through to the public verifier.

```bash
pnpm install
pnpm dev
```

Configure via `NEXT_PUBLIC_LEDGERPROOF_API_BASE` (defaults to `https://api-eu.ledgerproofhq.io`).

Apache-2.0.
