## Why LedgerProof

The EU AI Act's Article 50 enforces on **2 August 2026**. Penalties reach
**€15 million or 3% of global revenue**. Article 50(2) requires AI-generated
content to be marked in machine-readable format. LedgerProof is the open
protocol that does this — and it works in three lines of code.

<CardGrid stagger>
  <Card title="Python" icon="seti:python">
    ```python
    import openai, ledgerproof
    ledgerproof.attach(openai_client, publisher_id="LEI:...",
                       deployer_country="DE", deployer_name="Acme")
    # Every chat completion auto-issues an LPR receipt.
    ```
  </Card>
  <Card title="TypeScript" icon="seti:typescript">
    ```ts
    import OpenAI from "openai";
    import { attach } from "@ledgerproof/sdk/adapters/openai";
    attach(new OpenAI(), { publisherId: "LEI:...", deployerCountry: "DE",
                            deployerName: "Acme" });
    ```
  </Card>
  <Card title="Vercel AI SDK" icon="seti:typescript">
    ```ts
    import { ledgerproof } from "@ledgerproof/vercel-ai";
    streamText({ ..., experimental_telemetry: ledgerproof({ ... }) });
    ```
  </Card>
