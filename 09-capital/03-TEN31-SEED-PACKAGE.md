## I. What LedgerProof Is

LedgerProof is not a crypto company.
It does not issue a token.
It does not custody assets.
It does not speculate on price.

LedgerProof uses Bitcoin exactly as Satoshi designed it to be used:
as a permanent, neutral, globally-accessible timestamp authority
that no government, corporation, or adversary can rewrite.

Every LedgerProof Receipt (LPR) is three things:

1. **A SHA-256 hash** of the document — a mathematical fingerprint
   that changes if even one character of the document changes.

2. **An Ed25519 signature** — binding the claim to a specific
   publisher's key, so origin is cryptographically proven.

3. **A Bitcoin OP_RETURN commitment** — the hash is aggregated into
   a Merkle tree and the root is written permanently into a Bitcoin
   block, where it cannot be altered, deleted, or forged.

The receipt is publicly verifiable by anyone, forever, without
contacting LedgerProof. It requires no trust in us.
It requires only trust in Bitcoin.

**This is Bitcoin doing what Bitcoin does best —
in the one place no one else has built it:
the institutional document layer.**
